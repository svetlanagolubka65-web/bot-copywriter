## Конкурент 1: SMMplanner (AI-ассистент)

**Сайт:** smmplanner.com  
**Тип:** Веб-сервис + Telegram-интеграция

### Какую задачу решает
Планирование и автопостинг контента во все соцсети включая Telegram. ИИ-ассистент на базе ChatGPT помогает генерировать тексты постов по запросу.

### Какие функции есть
- Автопостинг по расписанию в 10+ соцсетей
- AI-ассистент: генерация постов, перефразировка, улучшение текста
- RSS-репостер (автоматически подхватывает новые статьи из RSS)
- Встроенный графический редактор «Холст»
- Аналитика постов и охватов
- Кросспостинг с UTM-метками
- Командная работа над контентом

### Примеры типичных диалогов
Пользователь заходит в веб-интерфейс → вставляет тему → ИИ генерирует текст поста → пользователь редактирует → планирует публикацию на время.

### Что делает его полезным
Полный цикл от написания до публикации в одном инструменте. RSS-репостер хорошо работает для новостных каналов.

### ✅ Что взять для нашего бота
- Идея RSS-мониторинга источников по ИИ-тематике
- Логика «сгенерировал → отредактировал → запланировал»
- Кнопки управления прямо в Telegram (вместо перехода в веб)

## Конкурент 2: Postmypost

**Сайт:** postmypost.io  
**Тип:** Веб-сервис с Telegram-интеграцией

### Какую задачу решает
Управление постингом в Telegram-каналах и соцсетях с командной работой и AI-генерацией контента.

### Какие функции есть
- Планирование постов с визуальной лентой контента
- AI-генерация текстов: идеи, переписывание, адаптация стиля
- Улучшение постов: сделать более вирусными, добавить CTA
- Репосты и импорт из RSS
- Подключение нескольких каналов
- Видеосообщения и опросы
- Аналитика (обновляется раз в сутки)

### Примеры типичных диалогов
«Напиши пост на тему ChatGPT для предпринимателей» → бот генерирует вариант → можно попросить «сделай более вирусным» → публикация.

### Что делает его полезным
Визуальный контент-план удобен для регулярных авторов. Есть функция улучшения уже написанных постов.

### ✅ Что взять для нашего бота
- Функция «улучши этот пост» (сделать вирусным / добавить CTA)
- Визуальный контент-план на неделю
- Возможность адаптации поста под разные форматы

## Конкурент 3: ChatPost Bot (@ChatPost)

**Тип:** Telegram-бот  
**Ниша:** Контент для блогеров и экспертов

### Какую задачу решает
Превращает идеи, голосовые сообщения, ссылки на статьи и YouTube-видео в готовые посты для Telegram/соцсетей.

### Какие функции есть
- Генерация постов из голосового сообщения (надиктовал → получил пост)
- Пост из ссылки на статью или YouTube-видео
- Контент-план на неделю по теме
- Сценарии для Reels/Shorts
- Генерация изображений к постам
- Разные стили: экспертный, личный, сторителлинг

### Примеры типичных диалогов
```
Пользователь: [голосовое сообщение 30 сек об ИИ]
Бот: Вот пост для Telegram:
"Сегодня протестировал новый инструмент..."
[кнопки: ✏️ Переписать | 📋 Скопировать | 🔄 Другой стиль]
```

### Что делает его полезным
Очень низкий порог входа — не надо ничего печатать, просто наговорил идею. Идеально для занятых авторов.

### ✅ Что взять для нашего бота
- **Голосовой ввод** — Света может наговорить комментарий к новости голосом
- Кнопка «Другой стиль» для перегенерации
- Генерация из YouTube-видео (взял ролик конкурента → адаптировал для своего канала)

## Конкурент 4: n8n Daily AI News Digest Workflow

**Источник:** n8n.io/workflows  
**Тип:** Автоматизированный workflow (шаблон)

### Какую задачу решает
Полностью автономный агент: каждое утро в 8:00 собирает новости об ИИ из RSS и Google News, фильтрует за последние 24 часа, суммаризирует через GPT-4 и отправляет дайджест в Telegram.

### Какие функции есть
- RSS Feed Aggregation из Google News + Hacker News
- AI-фильтрация: только релевантные ИИ-новости
- GPT-4 суммаризация с ключевыми инсайтами
- Отправка в Telegram-канал или личку
- Настраиваемое расписание (по умолчанию 8:00)
- Мульти-источниковое объединение (до 40 статей → топ-15)

### Примеры типичных диалогов
Система полностью автономна — пользователь получает утром сообщение:
```
📰 Дайджест ИИ-новостей — 17 июня
1. OpenAI выпустила GPT-5... [краткое резюме]
2. Google анонсировала... [краткое резюме]
...
```

### Что делает его полезным
Готовый технический шаблон, который работает без программирования. Легко адаптировать под любую тему.

### ✅ Что взять для нашего бота
- **Архитектуру сбора новостей** (RSS + несколько источников + фильтр за 24ч)
- Логику приоритизации: из 40 статей выбирать топ-3
- Отправку утреннего дайджеста до генерации постов

## Конкурент 5: AI-бот из vc.ru (Бесплатный AI-копирайтер по новостям)

**Источник:** vc.ru/telegram/2050576  
**Тип:** Telegram-бот (независимый разработчик)

### Какую задачу решает
Мониторит выбранные Telegram-каналы и X/Twitter, и автоматически перерабатывает каждую новую публикацию в пост в стиле автора через Gemini AI.

### Какие функции есть
- Подключение любых Telegram-каналов как источников
- Подключение X (Twitter) аккаунтов
- Персональный промпт: задаёшь стиль и формат итогового поста
- Фильтрация через «нет сигнала» — если нет нужного контента, бот молчит
- Предпросмотр перед отправкой в канал
- Поддержка 100+ языков источников

### Примеры типичных диалогов
```
Источник: пост в канале конкурента
→ Бот обрабатывает через Gemini
→ Генерирует пост в стиле автора
→ Отправляет на проверку в личку
[Кнопки: ✅ Опубликовать | ✏️ Изменить | ❌ Пропустить]
```

### Что делает его полезным
Мониторит конкурентов в реальном времени — ничего не пропустишь. Фильтрация «нет сигнала» убирает нерелевантный шум.

### ✅ Что взять для нашего бота
- **Мониторинг Telegram-каналов конкурентов** как источников (не только RSS)
- Механизм «нет сигнала» для фильтрации нерелевантного контента
- Персональный промпт с голосом автора (сохранённый один раз)

## Конкурент 6: n8n AI Telegram Bot Agent (Content Summarizer)

**Источник:** n8n.io/workflows/4457  
**Тип:** n8n workflow — умный ИИ-агент в Telegram

### Какую задачу решает
Telegram-бот, который по команде суммаризирует любую статью/URL, генерирует изображения и отвечает как персональный ИИ-ассистент — доступен 24/7.

### Какие функции есть
- `/summary <ссылка>` — суммаризация любого URL в 10-12 пунктов
- `/img <описание>` — генерация изображения
- `/help` — инструкция по командам
- Умная маршрутизация запросов (Switch node)
- Работает с несколькими пользователями одновременно
- Развёртывание за 15 минут

### Примеры типичных диалогов
```
Пользователь: /summary https://openai.com/news/...
Бот: 📝 Краткое содержание:
1. OpenAI анонсировала...
2. Ключевые изменения...
...
[Прочитать полностью →]
```

### Что делает его полезным
Позволяет быстро «переварить» любую статью и превратить её в основу для поста. Готов к запуску без программирования.

### ✅ Что взять для нашего бота
- Команда `/digest <ссылка>` — быстрый разбор статьи для поста
- Архитектура команд (/news, /post, /plan, /idea)
- Inline-кнопки для быстрого управления

## 🚀 Ключевые выводы для НейроСвета-агента

### Что есть у всех конкурентов (базовый минимум):
1. Сбор из RSS или ссылок
2. ИИ-генерация текста
3. Планирование публикации

### Чего НЕТ ни у кого (наше конкурентное преимущество):
1. **Мониторинг именно Telegram-каналов конкурентов** + фильтрация + адаптация в голосе автора — в одном боте
2. **Голосовой ввод комментария** к автоматически найденной новости
3. **3 формата на выбор** (новость / лайфхак / история+продажа) из одного инфоповода
4. **Мягкая продажа курса** вшита в шаблоны постов изначально, а не добавляется вручную

### Технические решения для реализации:
- **Оркестратор:** n8n (self-hosted) или Make.com
- **Источники:** RSS (Hugging Face, OpenAI, Google AI) + Telegram Parser (Telethon) + Twitter/X API
- **ИИ:** Claude API или GPT-4o (промпт с голосом НейроСветы)
- **Хранилище промпта и архива:** Google Sheets или Notion
- **Интерфейс:** Telegram Bot API (python-telegram-bot)
