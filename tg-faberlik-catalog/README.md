# Telegram Mini App — Каталог Фаберлик

**Статус: концепция** — код не написан, есть только исследование.

## Идея

Персональный брендированный каталог для консультантов Фаберлик в виде Telegram Mini App. Каждый консультант получает свою мини-версию каталога с:
- Живым каталогом (автосинхронизация цен)
- Записью на консультацию
- Нативной оплатой через Telegram Payments
- Клиентской базой (история покупок)
- Отзывами с фото результата

## Зачем

Официальный инструментарий Фаберлик (PDF-каталог, мобильное приложение) не позволяет персонализацию. Консультанты используют неофициальные боты — устаревшие, без синхронизации с ценами, без CRM.

## Стек (планируемый)

| Компонент | Технология |
|-----------|-----------|
| Frontend | React + Telegram WebApp SDK |
| Backend | Node.js / Python FastAPI |
| БД | PostgreSQL или Firebase |
| Оплата | Telegram Payments + YooKassa |
| Каталог | Фаберлик API (если доступен) или ручное обновление |

## Обязательные экраны MVP

1. Главная (профиль консультанта + баннер акции)
2. Каталог (категории + фильтры)
3. Карточка товара (фото, цена, описание)
4. Корзина
5. Оформление заказа
6. Подтверждение
