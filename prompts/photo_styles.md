# Промпты для AI-фотосессий (услуга «visual» в intake_bot.py)

Готовые промпты для Gemini (или другого генератора фото) под 5 пресетных стилей из бота-консультанта.
Когда клиент оформляет заказ через бота, полный текст промпта нужного стиля приходит в чат уведомлений
(`NOTIFY_CHAT_IDS`) вместе с фото клиента — остаётся скопировать промпт и вставить в Gemini.

**Источник истины — `PHOTO_STYLES` в `intake_bot.py`.** Этот файл — читаемая копия для правки текста
без кода. При изменении промпта обновляй оба места.

## ✍️ Свой стиль (когда клиент описывает сам)

Если клиент выбирает «Своё описание», бот вставляет его текст в общий шаблон вместо готового пресета:

```
Сгенерируй фотореалистичное фото человека по референсному фото — {описание клиента}. Ракурс — фронтальный
с лёгким поворотом в три четверти, имитация съёмки со смартфона. Естественные макияж и причёска, поза
непринуждённая, выражение лица живое. Освещение мягкое, естественное, подчёркивающее фактуру кожи и
одежды.
vertical 9:16, smartphone photo, realistic environment, ultra realistic skin texture. Сохрани черты лица
и идентичность без изменений, не изменяй форму лица, нос, губы, глаза и структуру кожи.
```

Соответствует `CUSTOM_PROMPT_TEMPLATE` в `intake_bot.py`.
